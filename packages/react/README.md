React wrapper for oplayer. (NOT WORKING ON REACT STRICT MODE)

## Install

```bash
npm i @oplayer/core @oplayer/react
```

const plugins = [

    pictureInPicture: true,
    slideToSeek: 'always',

  }),
  hls()
]
